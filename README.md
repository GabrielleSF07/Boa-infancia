#  Boa Infância

Site institucional desenvolvido para apresentação de serviços de recreação infantil e solicitação de orçamento online.

##  Sobre o Projeto

O projeto **Boa Infância** foi criado com o objetivo de divulgar serviços de recreação para festas infantis, como:

-  Caça ao Tesouro
-  Piquenique
-  Pintura em Tela

O site conta com:

- Página inicial
- Galeria de fotos
- Formulário de orçamento personalizado
- Layout moderno e responsivo
- Botões interativos e estilização personalizada

---

##  Tecnologias Utilizadas

- HTML5
- CSS3

---

##  Funcionalidades

✔ Navegação com menu fixo  
✔ Layout organizado com Grid e Flexbox  
✔ Botões personalizados  
✔ Checkbox estilizado como botão  
✔ Campo de observações expandido  
✔ Efeito hover em botões  
✔ Footer com informações e redes sociais
